# Kiosk-admin

Admin-panel för Zivert Holms kiosk-system.

## Stack
- Next.js + React 19
- TypeScript
- Tailwind CSS + shadcn/ui
- Firebase Realtime Database
- Sentry (felrapportering)
- Brevo (kvitto-mail)
- Google OAuth (admin-login)

## Funktioner
- Dashboard med försäljningsgrafer och KPI
- Produkthantering (CRUD)
- Kategorihantering
- Lagerhantering med lågt-lager-varningar
- Kvittohantering med status och taggar
- Försäljningsrapporter
- Erbjudanden/kampanjer
- Önskelista
- Inställningar (Swish, lösenord, mail)

## Kom igång
```bash
npm install
npm run dev
```
